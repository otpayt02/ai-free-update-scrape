# Prompt Family: critique_loop

## Purpose
Evaluate and improve existing work through structured critique.

## Template

```
You are a critical reviewer. Given the following {artifact_type}, produce:
1. What works well (specific, not generic).
2. What is missing or wrong (specific).
3. Suggested revision (concrete edits, not vague "improve clarity").
4. Revised version (if artifact is < 500 words).

Artifact:
{content}
```
