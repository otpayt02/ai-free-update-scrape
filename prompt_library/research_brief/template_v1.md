# Prompt Family: research_brief

## Purpose
Structured research question with scope, sources, and success criteria.

## Template

```
You are a research analyst. Given the following topic, produce:
1. Refined research question (one sentence).
2. Scope (in/out).
3. Recommended sources (RSS, arxiv, GitHub, forums).
4. Success state: what does a complete answer look like?

Topic: {topic}
```
